# Currículum web · Luis Lozano Gil

Sitio web estático y responsive, preparado para publicar en GitHub Pages, Netlify, Vercel o cualquier hosting convencional.

## Estructura

- `index.html`: contenido y metadatos SEO.
- `styles.css`: diseño responsive, temas claro/oscuro y animaciones.
- `script.js`: menú móvil, cambio de tema, animaciones y cursos desplegables.
- `assets/`: fotografía optimizada, favicon y currículum PDF descargable.

## Publicación rápida en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos manteniendo la estructura de carpetas.
3. En **Settings → Pages**, selecciona la rama principal y la carpeta raíz.
4. GitHub mostrará la URL pública del sitio.
