# Currículum web · Luis Lozano Gil

Sitio web estático y responsive, para mostrar mi curriculum ;) .

## Estructura

- `index.html`: contenido y metadatos SEO.
- `styles.css`: diseño responsive, temas claro/oscuro y animaciones.
- `script.js`: menú móvil, cambio de tema, animaciones y cursos desplegables.
- `assets/`: fotografía optimizada, favicon y currículum PDF descargable.
